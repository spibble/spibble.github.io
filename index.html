currently in progress:

  • CCE 3 project website: for one of my classes i got the opportunity to make a website, and the theme of the class was community engagement, so i figured why not make a website for a school club that aims to build community? i'm finishing up the first version of the website right now, and i've learned a lot about making websites from this project (the last one i made was three years ago and basically raw HTML and CSS). on my more fully-featured personal website, i'd like to have a space to write blog posts and this will probably be the subject of the first one!

projects i've completed:

  • robotic arm: this arm is powered by an esp-32 and has 6 degrees of freedom using some SG90 and MG996R servo motors. this was a super fun project because it was my second time using a microcontroller, my first time using a breadboard, wires, and other electronic components for a real project, and i also got to hone some of my existing skills! as a stretch i also got a simple web interface working using a websocket connection, which was also new to me! unfortunately i didn't get as much experience with IK or lower-level programming as i wanted to, but since i kept the arm...
  • lost device tracker: this was my CSE 190 project! it's an stm32 dev board project that monitors accelerometer readings and can activate "lost mode" after the device hasn't moved for a while, in which it can connect to a phone and relay information like how much time has passed. this was my very first embedded project, my first time working with an MCU, etc. and it was really fun! i feel like i learned a lot of the basics like embedded C concepts, ISRs, GPIO, and I2C. it was also really fun optimizing power; we were able to get it down to around 80 uA, which was unfortunately not enough to win (though we did get second).
  • url shortener: to be honest, i'm not super proud of this one, but it is a thing that i made so it goes in here! it's a url shortener and it does exactly that; you plug in a url, get a shortened one, plug in the short url, and it redirects you to the correct page. this was an opportunity to learn basic backend dev using python/fastapi/sqlite, though i would definitely be interested in re-making this project again in the future to make it more of an actual project rather than something i just shat out super quickly lol

note: these lists are not exhaustive; i have just put here what i can remember but i plan to add more!
